# Yamashita.Features
画像特徴量記述子を抽出するためのライブラリです。  
HOGは勾配特徴，フーリエ変換は周期的な特徴を記述するのに利用するとよいでしょう。  
フーリエ変換は1次元も実装しており，画像に限らず音声やノイジーな信号処理に応用できます。
